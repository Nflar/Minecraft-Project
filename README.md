(2) • Triggerbot - F6 to enable/disable, only borderless + crosshair that goes red when hover on enemy ( it has to have red in the center for it to correctly detect )

#Функции

(1) • Вор сундуков (Chest-stealer) — Shift + Левый Alt мгновенно забирает лут из маленького сундука (любое стандартное разрешение Windows, без кастомных)

(2) • Триггербот (Triggerbot) — F6 для включения/выключения, работает только в оконном режиме без рамок + прицел должен становиться красным при наведении на врага (в центре должен быть красный цвет для корректного определения)
